# Void Blaster: Neon Miko — Vaporwave Shrine Edition

A high-performance, retro-inspired Vaporwave bullet-hell game written in C++ using Raylib.

## 🚀 Overview
Void Blaster is a fast-paced "shmup" (shoot-em-up) with a unique aesthetic, featuring synthwave colors, multiple ship archetypes, and intense enemy waves.

### ✨ Technical Highlights
*   **High Performance:** Uses pre-allocated bullet pools (`BULLET_POOL_SIZE = 1024`) to eliminate heap fragmentation during gameplay.
*   **Modular Design:** Cleanly separated architecture across `draw.cpp`, `game.cpp`, `hud.cpp`, `vfx.cpp`, and `world.cpp`.
*   **Custom Graphics:** Built on [Raylib](https://www.raylib.com/), leveraging modern C++ interactive systems.

## 🛠 Building the Game

### Prerequisites
*   C++ Compiler (GCC/Clang)
*   [Raylib](https://github.com/raysan5/raylib) installed on your system.

### Build Instructions
Run the provided build script or use the Makefile:
```bash
./build.sh
# or
make
```
